About Carry The One

What is Carry The One?

Carry The One is a UIL Number Sense trainer designed to help students master mathematical tricks and techniques. This tool provides practice problems and explanations for various number sense concepts including geometry, multiplication tricks, and algebra.

Whether you're preparing for UIL competitions or just want to improve your mental math skills, Carry The One offers a comprehensive learning experience with detailed explanations and interactive practice.

Contributing to the Project

We welcome contributions! Here's how you can help improve Carry The One:

  1. Fork the repository on GitHub to create your own copy
  2. Create a new branch for your feature or fix: git checkout -b feature/your-feature-name
  3. Make your changes - add new tricks, improve explanations, or fix bugs
  4. Test your changes to ensure everything works correctly
  5. Commit your changes with a clear message: git commit -m "Add new multiplication trick"
  6. Push to your fork and create a Pull Request on GitHub
  7. Wait for review - we'll review your PR and provide feedback

For questions or suggestions, please open an issue on the GitHub repository.

Formatting Markdown Pages

When creating or editing trick explanation pages, follow this format: 

# Trick Name

## Formula
The formula or key concept here.

## Quick Trick
Step-by-step explanation.

## Example
Worked example with solution.

## Visual Tip
Helpful mnemonic or visual aid.

## Practice Problems
1. Problem 1 → Answer
2. Problem 2 → Answer

MathJax Support

Use MathJax for mathematical expressions:

  • Inline math: $x^2 + y^2$
  • Display math: $$A = \frac{1}{2} \times base \times height$$

File Location

Place markdown files in the tricks/ directory and update explanationIDs.json with the mapping.

Understanding Trick Explanations

Formula

The mathematical formula or rule that the trick is based on. This is the foundation you need to understand.

Quick Trick

Step-by-step instructions on how to apply the trick quickly. This is the mental shortcut you'll use in competitions.

Example

A worked example showing the trick in action. Study these carefully to understand the pattern.

Visual Tip

Memory aids, patterns, or visual cues to help you remember the trick. These are especially useful for quick recall.

Practice Problems

Additional problems to test your understanding. Try solving these without looking at the answer first!
