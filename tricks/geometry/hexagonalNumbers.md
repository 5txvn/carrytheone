# Hexagonal Numbers
## Notes
None.
## General Explanation
Similar to both triangular and pentagonal numbers, hexagonal numbers look like this mathematically:
![Hexagonal Numbers Visualization](https://i0.wp.com/math1089.in/wp-content/uploads/2021/05/image-17.png?resize=598%2C210&ssl=1)<br>
The nth hexagonal number can be found using the formula $H_n=2n^2-n$
## Example 1
*Find the 8th hexagonal number.*<br>
To find the 8th hexagonal number, we can use the formula to get $H_8=2\times 8^2-8=120$.
