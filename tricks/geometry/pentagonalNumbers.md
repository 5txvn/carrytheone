# Pentagonal Numbers
## Notes
None.
## General Explanation
Similar to triangular numbers, pentagonal numbers look like this mathematically:
![Pentagonal Numbers Visualization](https://media.geeksforgeeks.org/wp-content/uploads/20241021123936538334/Pentagonal-Number.webp)<br>
The nth pentagonal number can be found using the formula $P_n=\frac{3n^2-n}2$
## Example 1
*Find the 8th pentagonal number.*<br>
To find the 8th pentagonal number, we can use the formula to get $P_8=\frac{3\times 8^2-8}2=92$.
