# Converting Base b to Base 10
## Notes
None.
## General Explanation
1. Multiply the first digit of the number by b
2. Add the next digit
3. Multiply the result by b
4. Continue until there are no more digits to add
## Example 1
$$1234_5=$$
1. Take the first digit (1), and multiply it by the base ($1\times 5=5$)
2. Add the next digit, 2 ($5+2=7$)
3. Multiply the result by the base ($7\times 5=35$)
4. Add the next digit, 3 ($35+3=38$)
5. Multiply the result by the base ($38\times 5=190$)
6. Add the next/last digit, 4 ($190+4=194$)
7. Final Answer: **194**
## Example 2
$$583_9=$$
1. Take the first digit (5), and multiply it by the base ($5\times 9=45$)
2. Add the next digit, 8 ($45+8=53$)
3. Multiply the result by the base ($53\times 9=477$)
4. Add the next/last digit, 3 ($477+3=480$)
5. Final Answer: **480**
