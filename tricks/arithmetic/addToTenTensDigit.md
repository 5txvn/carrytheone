# Add to Ten Tens Digit
## Notes
+ Right-to-left trick
## General Explanation
This trick is used to multiply two 2-digit numbers that have the same last digits and have first digits that add to 10:
1. Multiply the last digits by each other (or technically, square it). Write this result on the right side of the answer blank.
2. Multiply the first digits by each other and add the last digit. Write this result to the left of the current answer.
## Example 1
$$73\times 33=$$
1. Multiply the last two digits ($3\times 3=09$). Write this number on the right side of the blank (**09**)
2. Multiply the first two digits ($7\times 3=21$) and add the last digit ($21+3=24$). Write this result to the left of the current answer (**2409**)
3. Final Answer: **2409**
## Example 2
$$86\times 26=$$
1. Multiply the last two digits ($6\times 6=36$). Write this number on the right side of the blank (**36**)
2. Multiply the first two digits ($8\times 2=16$) and add the last digit ($16+6=22$). Write this result to the left of the current answer (**2236**)
3. Final Answer: **2236**
