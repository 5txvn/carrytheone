# $\frac 1a+\frac 1x=\frac 1b$
## Notes
+ Simplification required
## General Explanation
This is another fraction addition problem that appears frequently on number sense, and we can derive a formula for x using some basic algebra:
1. Move $\frac 1a$ to the other side to get $\frac 1x=\frac 1b-\frac 1a$
2. Simplify the right side using the butterfly method to get $\frac 1x=\frac{a-b}{ab}$
3. Take the reciprocal of both sides to get a final answer of $x=\frac{ab}{a-b}$
## Example 1
*What is x if $\frac 13+\frac 1x=\frac 17$?*
1. Since $a=3$ and $b=7$, using the formula $x=\frac{ab}{a-b}$ we get $x=\frac{3\times 7}{3-7}=-\frac{21}4$
2. Final Answer: **$-\frac{21}4$**
## Example 2
*What is x if $\frac 1{12}+\frac 1x=\frac 15$?*
1. Since $a=12$ and $b=5$, using the formula $x=\frac{ab}{a-b}$ we get $x=\frac{12\times 5}{12-5}=\frac{60}7$
2. Final Answer: **$\frac{60}7$**
