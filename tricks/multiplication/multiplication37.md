# Multiplication by 37
## Notes
+ Number must be divisible by 3
## General Explanation
We can use the fact that $37\times 3=111$ to our advantage for this trick:
1. Divide the number by 3
2. Multiply by 111 (refer to multiplication by 111 trick)
## Example 1
$$27\times 37=$$
1. $27\div 3=9$
2. $9\times 111=999$
## Example 2
$$84\times 37=$$
1. $84\div 3=28$
2. $28\times 111=3108$
