# Multiplication by 143
## Notes
+ Number must be divisible by 7
## General Explanation
We can use the fact that $143\times 7=1001$ to our advantage for this trick:
1. Divide the number by 7
2. Multiply by 1001
## Example 1
$$28\times 143=$$
1. $28\div 7=4$
2. $4\times 1001=4004$
## Example 2
$$98\times 143=$$
1. $98\div 7=14$
2. $14\times 1001=14014$
